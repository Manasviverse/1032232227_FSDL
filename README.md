# Flight Booking Management System

A MERN stack CRUD application for managing passenger flight booking details.

## Features
- Insert passenger details
- Search passenger records by phone number
- Update passenger details by phone number
- Delete passenger records by phone number
- View all records in tabular format

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB + Mongoose

## Project Structure
- `client/` React frontend
- `server/` Express backend

## Setup
1. Make sure MongoDB is installed and running locally on `mongodb://127.0.0.1:27017`.
2. From the project root, install dependencies:
   - `npm run install-all`
3. Start the app:
   - `npm run dev`
4. Open the frontend URL shown by Vite, usually `http://localhost:5173`.

## Environment Files
- `server/.env`
- `client/.env`

Default values are already provided for local development.
